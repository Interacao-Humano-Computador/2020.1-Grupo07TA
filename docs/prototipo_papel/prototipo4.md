# Protótipo de papel - Baixar formulário para reclamação

## Participante
- Bruna Almeida Santos


<p align="justify">O protótipo de papel foi desenvolvido no dia 12 de novembro de 2020, com base na funcionalidade de baixar um formulário para reclamação junto à Prefeitura Municipal de Águas Lindas de Goiás, disponível no <a href="https://aguaslindasdegoias.go.gov.br/">site</a> da mesma.</p>
