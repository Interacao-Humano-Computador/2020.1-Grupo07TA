# **Grupo 07 - Prefeitura de Águas Lindas de Goiás**

<p align = "justify">Repositório destinado ao Projeto da matéria de Interação Humano-Computador, ministrada pelo Professor Dr. André Barros de Sales, da Universidade de Brasília, no semestre de 2020.1.</p>
<p align = "justify">O projeto foi desenvolvido entre os dias 07 de setembro e 01 de dezembro do ano de 2020.</p>

<p align = "justify">Nosso objetivo é analisar o site da  <a href="https://aguaslindasdegoias.go.gov.br/">Prefeitura da cidade de Águas Lindas de Goiás</a> , de acordo com os assuntos abordados ao longo de todo o semestre. Detalhes sobre o site e uma breve avaliação do mesmo podem ser visualizados <a href="https://interacao-humano-computador.github.io/2020.1-Prefeiturade-Aguas-Lindas-de-Goias/planejamento/site/">aqui</a>.</p>

# Ferramentas

<p align = "center"><img src="./imagens/github.png" width="250">
		<p align="center">O Github é utilizado para o versionamento e armazenamento de todos os documentos que serão produzidos pelo grupo. Além disso, a nossa Wiki está hospedada na plataforma GitPages.</p>
</p>

</br>

<p align = "center"><img src="./imagens/telegram.png" width="250">
		<p align="center">O Telegram é utilizado para a facilitar a comunicação entre os membros do grupo. Por lá, são marcadas as reuniões e todas as dúvidas pontuais que os integrantes tem são respondidas prontamente.</p>
</p>

</br>

<p align = "center"><img src="./imagens/meet.png" width="250">
	<p align="center">A plataforma Google Meet foi utilizada para as reuniões, tanto para a divisão das tarefas de cada entrega, quanto para as gravações das apresentações.</p>
</p>

</br>

<p align = "center"><img src="./imagens/trello.png" width="250">
  <p align="center">A ferramenta de organização Trello foi utilizada para a coordenação de tarefas para cada integrante. A cada entrega, as tarefas eram organizadas no Trello, com participante definido e data que a tarefa deveria ser entregue.</p>
</p>

</br>

<p align = "center"><img src="./imagens/figma.png" width="400">
  <p align="center">A plataforma Figma foi utilizada para o desenvolvimento dos <a href="https://interacao-humano-computador.github.io/2020.1-Prefeiturade-Aguas-Lindas-de-Goias/prototipo_alta_fidelidade/prototipo_alta_fidelidade_principal/#resultados">protótipos de alta-fidelidade</a>, na fase final do nosso projeto.</p>
</p>
